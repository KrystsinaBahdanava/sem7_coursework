Data - все данные для быза данных. Например, excel файл со списком улиц Минска.
DB - скрипты БД
myapp - сайт
диаграммы - тут лежат все диаграммы для записки: UML, IDEF0

Инструкция по запуску: 
myapp->launch server.bat
Перейти по адресу http://localhost:3030

База данных - MySQL. Скрипты создания всех таблиц лежат в папке DB, выгруженные данные в папке Data.
